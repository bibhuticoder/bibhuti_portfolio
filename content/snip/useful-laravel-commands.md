## Migrations
```bash
php artisan migrate:refresh
php artisan db:seed
php artisan migrate:refresh --seed # migrate + seed
php artisan migrate:reset
```

## Create Request

```bash
php artisan make:request StoreBlogPost
```

## Composer
```bash
composer dump-autoload
```
